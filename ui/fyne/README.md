# Desktop UI (Wails)

Desktop UI is implemented with Wails.

Backend entrypoint:
- `cmd/npcgen-wails`

Frontend assets:
- `ui/wails/dist`
- Build guide: `ui/wails/README.md`

Run entrypoint:
- `go run ./cmd/npcgen-wails`

Optional data directory override:
- `go run ./cmd/npcgen-wails --data-dir "G:\\My Drive\\RootProject\\NPCgenGo\\data"`
